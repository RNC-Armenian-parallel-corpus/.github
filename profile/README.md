# Проект по пополнению Армянского параллельного корпуса НКРЯ
Привет!
Мы команда магистрантов-копьютерных лингвистов НИУ ВШЭ (г.Москва), и цель нашего проекта - пополнить [Армянский параллельный корпус НКРЯ](https://ruscorpora.ru/results?search=CoQBEmQKYgoUCgtsYW5nX3NlYXJjaBIFCgNydXMSSgoJCgNsZXgSAgoACgoKBGZvcm0SAgoACgsKBWdyYW1tEgIKAAoJCgNzZW0SAgoACgwKBnRyYW5zbBICCgAKCwoFZmxhZ3MSAgoAKhAKCAgAEAoYMiAKIABABXgBMgcIBRIDaHllOgEB).

На данный момент (июнь 2025г.) корпус включает 28 текстов, или 1.5 млн токенов. Мы проделали большую работу и собрали и обработали более 1500 новых пар параллельных текстов. 

В этом проекте собраны основные скрипты и библиотеки, которые мы разработали в ходе проекта.
